# ToriSidebar v0.1.9

🐦 A lightweight Edge Sidebar replacement for Windows.

---

## English

### What's New in v0.1.9

**New Features**
- **Full English & Chinese language support** — The entire UI, settings panel, modals, tray menu, and injected navigation bar are now bilingual.
- **Language switcher in Manage Mode** — Click ⚙️ to expand the bar and switch between English and 中文 instantly. Preference is persisted across restarts.
- **English as default language** — First-time users will see the app in English out of the box.

**Fixed**
- Tray menu text is now consistently in English ("Quit").

### Download

Download `ToriSidebar.exe` from the release and double-click to run. No installation needed.

### Requirements

- Windows 10/11 (x64)
- WebView2 Runtime (pre-installed on Windows 10/11)

---

## 中文

### v0.1.9 更新内容

**新增功能**
- **完整的中英双语支持** — 全部 UI 界面、设置面板、弹窗、托盘菜单以及注入式导航栏均已支持双语切换。
- **管理模式内置语言切换** — 点击 ⚙️ 展开边栏后，可直接在「语言」选项中切换 English / 中文，偏好设置会自动记忆。
- **默认使用英文** — 首次打开应用即显示英文界面。

**修复**
- 系统托盘右键菜单统一为英文 "Quit"。

### 下载

从 release 区域下载 `ToriSidebar.exe`，双击即用，无需安装。

### 系统要求

- Windows 10/11（64位）
- WebView2 运行时（Win10/11 已预装）
